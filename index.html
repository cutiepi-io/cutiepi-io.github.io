擺脫束縛、帶著你的程式到處探險 // Your Raspberry Pi Projects, Untethered // 放飞自我，带上你的应用即刻去探险

CutiePi 是台一應俱全的樹莓派平板電腦，打開開關就能使用，所有你需要的創作工具都在裡面。帶上你的樹莓派專案，隨時隨地創造、盡情探索你的想法。 // CutiePi is an all-in-one Raspberry Pi tablet. Liberate your Pi project from the desk, and start creating wherever the idea strikes. // CutiePi 是一台开机即用，拥有必备的编程工具的一体化树莓派平板电脑。带上你的专属定制树莓派，随时随地创作，尽情探索、验证、实现你的奇思妙想。

五臟俱全的硬體功能 // The all-in-one hacking platform // 你想要的都在这里

CutiePi 將完整的樹莓派運作環境裝進平板的造型中，不再需要外接的螢幕與電源。由於使用了樹莓派的 Compute Module 模組，不但保有比開發板要纖瘦的身材，還增加了豐富的功能，並像一般電子產品一樣地攜帶與充電。 // CutiePi is a complete Raspberry Pi in a tablet form factor, minus the trouble of connecting monitor or power supply. It's slimmer because of using Compute Module, and you can carry or recharge it like your everyday gadgets. // CutiePi 将完整的树莓派运行环境装进平板中，无需外接屏幕和电源。小身板却不失性能的树莓派 Compute Module 模块，不仅使体积比树莓派开发板更小，还增加了丰富的功能，亦可像一般电子产品一样便携与充电。

真正可用的觸控介面 // The out-of-the-box experience // 爱不释手的触控界面

無論是打開熟悉的應用程式，遠端連線到伺服器工作，甚至是顯示自訂的儀表板，CutiePi 為了觸控螢幕量身打造的介面都能讓你輕易達成，也省去外接鍵盤滑鼠的必要。 // Whether you're opening an app, SSH-ing to a remote server, or displaying a dashboard, CutiePi's touch-friendly interface can help you with that. Connecting keyboards and mouse are no longer a necessity. // 不管是打开熟悉的应用软件，还是连接远程服务器，甚至显示自定的仪表盘，CutiePi 专属定制的触控界面都能让你轻松完成，省去了外接键盘鼠标的麻烦。

完全開放、任意改造 // Free to make it yours // 完全开放、任意定制

因為我們始終相信開放的力量，也相信人們該握有對自己使用科技的主控權，你看到的一切成果都是開放授權的 — 從操作介面、電路設計到外殼造型，CutiePi 所有的軟硬體設計都可按照你的想法自由取用、任意改造。 // We love and support open source, and we believe people should have control over the technology they use. Everything we made, including the schematics, PCB, drivers, firmware, and UI is open source. // 因为我们始终相信开放的力量，也相信人们应该拥有自己使用科技的权力，你看到的一切成果都是关于开放授权的 —— 从操作界面，电路设计到外壳形状，CutiePi 所有的软硬件设计都可按照你的想法自由取用、任意定制。

CutiePi board V1.0

專門為攜帶式樹莓派專案設計的嵌入式主板，強化了電源管理與鋰電池電量監測功能；並可以搭配 HDMI 或 MIPI DSI 螢幕使用。 // A custom CM3/CM3+ carrier board designed for portable use, with enhanced power management and Li-Po battery level monitoring features; works with selected HDMI or MIPI DSI displays. // 专门为便携式树莓派项目设计的嵌入式主板，加强了电源管理与锂电池电量监测功能；并可以搭配 HDMI 或 MIPI DSI 屏幕使用。
