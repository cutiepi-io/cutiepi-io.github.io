Your Raspberry Pi Projects, Untethered

CutiePi is a complete Raspberry Pi in a tablet. Liberate your Pi project from the desk, and start creating wherever the idea strikes.

We will notify you when it’s ready

The all-in-one creation kit

With the touchscreen and battery, there's no longer a need to connect a keyboard, monitor and power supply to set up your Pi.

The battery level reading and power management features allow you to carry or recharge it like your everyday gadgets.

Make portability easy

CutiePi shell, the mobile friendly UI, turns your Linux environment into a functional tablet while maintaining Raspbian OS compatibility. It provides everything you need on the go: from typing a command in terminal, connecting to a WiFi hotspot, to logging into webpages.

Take your existing projects for Pi, load it onto CutiePi, and run it right away. Make your project portable couldn't be easier.

See CutiePi shell in action

Because we care, we designed our own hardware.

And because we are cool and care even more, we’ve made everything open.

The CutiePi board

At the heart of the CutiePi is our custom PCB, the CutiePi board.
This CM3/CM3+ compatible carrier board comes with features for portable use, and is a proud OSWHA-certified open source hardware.

Check out the design files on GitHub

Frequently Asked Questions

What kind of apps are supported?

Other than built-in apps and UI designed for the touch screen, we hope to support Raspbian PIXEL apps via XWayland.

How's the development progress?

We have finished most of hardware enablements and driver verifications for EVT (CutiePi board V1.0), soon to start DVT and enclosure design.

What's the target price?

We'll know better in DVT stage, however it will likely be in the $150-$250 range. Please find the bill of materials in design files if you can't wait.

Do you plan to crowdfund?

We might, check out the blog post for more detail.

Any release date?

Most of the design files has been released, we will update the availability soon.
